# Examples

Available scripts in `examples/`:

- `all_events_to_txt.py`: log all events and persist changed payloads.
- `event_names_only.py`: print event names only.
- `clock_updates.py`: listen to `ClockUpdatedSeconds`.
- `goal_scored_typed.py`: typed `GoalScored` example.
- `configure_and_connect.py`: configure the exporter and connect with the same port.

Run from repository root:

```bash
python examples/event_names_only.py
```

For more practical snippets, see [Recipes](recipes.md).
